# Admin Web V41

The Business Outlets page now supports outlet drill-down. Each outlet card has a Full Dashboard button that opens:

`/business-outlets/{outletId}`

The drill-down dashboard includes:

- Date-range based summary.
- Total, online, COD, and offline sales.
- Average order value.
- Stock items and low-stock count.
- Day closing calendar.
- Manager/contact details.
- Best selling foods.
- Slow/non-performing foods.
- Current stock table.
- Stock update dialog.
- Stock movement/activity ledger.
- Exact outlet location editor.
- Outlet accounting CSV export.

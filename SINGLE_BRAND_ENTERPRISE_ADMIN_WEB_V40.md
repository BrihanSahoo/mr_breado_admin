# Mr Breado Admin Web v40

Admin web is converted toward a head-office business dashboard.

## New screen
`/business-outlets`

## Admin can
- Create Mr Breado outlets.
- Prepare backend enterprise schema.
- See total outlet sales, online sales, COD sales, orders.
- See outlet cards with low stock, best foods, slow foods.
- Set login credentials for outlet managers.
- Open outlet calendar/day-closing ledger.
- Export CSV accounting report for EzoBooks-style preparation.

## Navigation changes
- Marketplace restaurant wording is removed from primary nav.
- Business Outlets is the main section.
- Restaurant payout/verification/franchise links are no longer primary business flow.
